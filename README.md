# canp_integracao

Repositório separado para a integração com o sistema de venda.

Escopo planejado:
- Job de importação/sincronização de pedidos do sistema de venda
- Tela(s) de auditoria da integração (logs, status, falhas)
- Disparo manual de sincronização e reprocessamento

Configuração:
- Variáveis em `.env` (use `.env.example` como base)
